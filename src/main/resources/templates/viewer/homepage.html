About FunOlympic Games 2024 Web Application

I have selected the scenario of developing an online registration system for the FunOlympic Games” within the Computer System Engineering. This scenario focuses on creating a web application for users. System has two primary user roles: “Viewers” and “Admin”. The Admin, is a super user, is responsible for maintaining the online web application system. Admin can change passwords, deleting users, and enabling and disabling “Viewers” for system login. Viewers are normal users who can sign up and log in to the system using their email and password. Viewers have the ability to select multiple categories of games and to watch online broadcasts.

Technologies Used

For this project, I have used Java programming language, the Spring Boot framework with Thymeleaf templating, and MongoDB for the database.
